A Peek Inside the Headspring Toolbox

Oct 10th, 2012

Here is a peek into the Headspring toolbox.  I did a simple analysis of the tools used at Headspring by analyzing the Nuget package files for a subset of the projects we have in source control.  I created a PowerShell script to pull out the package names from the Nuget packages.config files.

Here is a list of items we have used to build projects for our clients over the last several years at Headspring. The tool list includes both brownfield and greenfield projects and is by no means comprehensive. I manually added items that I felt were noteworthy but didn’t show up in the Nuget packages.

Many of the tools listed would still be my weapon of choice if I started a new project today. I will follow up this post with a list of tools I would recommend for a greenfield ASP .NET MVC website.
