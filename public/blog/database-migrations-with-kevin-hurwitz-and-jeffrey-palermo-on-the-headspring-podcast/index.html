Database Migrations With Kevin Hurwitz and Jeffrey Palermo

In this cast, Kevin Hurwitz (Chief Architect) discusses database migrations and database change management with Jeffrey Palermo (President/COO) while covering the problem, options, and our recommended approach.  We use an open source tool called Tarantino in our deployment-based process.  This tool can be executed via a NAnt or Psake build script as well as stand-alone from the command line.  Here are some additional resources.

 

  • Continuous Integration book (see chapter 5 for full database coverage)
  • Screencast from several years ago demonstrating Tarantino and NAnt execution
  • Ryan Kelley’s blog post about using Tarantino
  • Tarantino database change management documentation
