Open Source

Open source software is a big part of what we do at Headspring. We often use OSS projects when building solutions and sometimes we end up creating open source projects from the work we do within a custom application.

Check out some of our popular open source libraries and projects:

AutoMapper

A convention-based object-object mapper. AutoMapper uses a fluent configuration API to define an object-object mapping strategy. AutoMapper uses a convention-based matching algorithm to match up source to destination values. Currently, AutoMapper is geared towards model projection scenarios to flatten complex object models to DTOs and other simple objects, whose design is better suited for serialization, communication, messaging, or simply an anti-corruption layer between the domain and application layer.

Add this library to your project via NuGet with Install-Package AutoMapper

  • AutoMapper on GitHub
  • AutoMapper

Enumeration

A utility class. Common uses include Java-like enumerations, dispatch table, flyweight, and drop-down list items/select options. No third-party dependencies.

Add this library to your project via NuGet with Install-Package Enumeration

  • Enumeration on GitHub

Formo

Formo lets you use your configuration file as a dynamic object. You can also strongly type your configuration variables and avoid repetitive casting code as well as configure with defaults up front and add in configured values after your project is already in production.

Add this library to your project via NuGet with Install-Package Formo

  • Formo on GitHub
  • Formo

Fixie

Fixie is a .NET convention-based test framework similar to NUnit and xUnit, but with an emphasis on low-ceremony defaults and flexible customization. Fixie gives you the ability to customize your tests in such a way that the framework is never in your way.

Add this library to your project via NuGet with Install-Package Fixie

  • Fixie on GitHub
  • Fixie

Tarantino

The Tarantino project is a collection of libraries and tools to facilitate change management and development.

  • Tarantino on GitHub

NAAK (.NET Accessibility Analysis Kit)

The .NET Accessibility Analysis Kit (NAAK) is designed to allow compliance with the US Rehabilitation Act, Section 508.

  • NAAK (.NET Accessibility Analysis Kit) on GitHub
